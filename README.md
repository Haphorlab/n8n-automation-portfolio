### [RAG-Powered Knowledge Base Chatbot](/Haphorlab/n8n-automation-portfolio/tree/main/rag-chatbot-system)

Intelligent document Q&A system using Retrieval-Augmented Generation for automated customer support and internal knowledge management.

**Key Features:**

* Automatic document processing from Google Drive
* Vector-based semantic search with Pinecone
* GPT-3.5 powered answer generation
* REST API webhook for easy integration
* Real-time knowledge base updates
* Multi-format document support (PDF, DOC, TXT)

**Tech Stack**: n8n, OpenAI (text-embedding-ada-002, GPT-3.5-turbo), Pinecone, Google Drive  
**Use Case**: Customer support automation, internal knowledge bases, FAQ systems
